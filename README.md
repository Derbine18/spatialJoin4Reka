# spatialJoin4Reka
cheminement étape par étape : 
- ouvrir QGIS et sa console python
- importer le fichier spatialJoin.py dans QGIS
- modifier les chemins d'accès des lignes 10, 14, 50 et 51 par votre chemin d'accès
- sauvegarder le programme et l'exécuter

les quelques évènements qui posent problème, sont les 3 fichiers pdf ci-joints :
le fichier rouge.pdf contient les évènements problématiques, des migrants morts en Amérique centrale et du Sud alors que la carte n'est censée que concerner les migrants morts dans les zones délimitées
le fichier rouge 2 contient cette fois les évènements problématiques des migrants morts à Madagascar et au Bangladesh
le fichier rouge erreur contient les évènements avec une erreur, ceux-ci sont censés se trouver au Maroc, en Lybie et dans les îles Canari mais au lieu de ça elles se trouvent en dehors de la carte
